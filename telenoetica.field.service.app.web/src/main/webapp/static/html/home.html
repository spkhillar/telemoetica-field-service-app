Home page is accessible for all user roles and contains 5 section:

1. Visit Analysis Chart (Last 7 Days): this is a bar chart representation of number of visits done during last 7 days for all 4 visits.

2. Routine Visit: Last 5 Routine visit summary (Site id and visit time).

3. Callout Visit: Last 5 Callout Visit summary (Site id and visit time).

4. Diesel Visit: Last 5 Diesel visit summary (Site id and visit time).

5. Maintenance Visit: Last 5 Maintenance visit summary (Site id and visit time).

All visit section has "View All" link, on clicking of which you can navigate to report section of that visit.
