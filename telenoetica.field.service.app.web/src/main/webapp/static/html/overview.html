Welcome to the one-stop source for help on Helios field application. Find everything you need for setting up, exploring features and usage information.

To download android application click on DOWNLOAD ANDROID APP link on top navigation.Download and install the application on your android phone

Uninstall the older version(if you already have it on your smart phone) before installing the new one and make sure you do not have any pending visit data which is saved on phone but not submitted on server.

If you are experiencing any difficulties in installation ,please contact system admin.

You may or may not have access to all the pages of web application depending on your user role.If you have any specific need to access pages which are not accesible for you, please contact system admin.

Android application can be used only by Field service Engineers(FSE) and Admin users, other users can not login to android application on smart phones.
