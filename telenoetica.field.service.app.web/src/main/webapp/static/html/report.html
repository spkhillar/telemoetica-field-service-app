This section contains 6 pages :

1. Routine Visit Report: This page contains list of all routine visits till date with filter, refresh and export to excel functions.

2. Callout Visit Report: This page contains list of all Callout Visit till date with filter, refresh and export to excel functions.

3. Diesel Visit Report: This page contains list of all Diesel Visit till date with filter, refresh and export to excel functions.

4. Maintenance Visit Report: This page contains list of all Maintenance Visit till date with filter, refresh and export to excel functions.

5. Diesel Details reports: This page contains list of Diesel Details reports generated in last 12 months and link to download the same. Diesel Details reports are generated on month end and send via email to configured mailing list.

6. Spares Utilization reports: This page contains list of Diesel Details reports generated in last 12 months and link to download the same. Spares Utilization reports are generated on month end and send via email to configured mailing list.

Other common buttons(functions) at the bottom of the page are:

+ Filter

Filter allow you to manage the list of information on grid, To create a filter

1. Click the filter button in your page. A window that allows you to specify your search criteria will appear.

2. Enter your search criteria. If you want to check that your search worked correctly, click the find button..

3. You can provide multiple search criteria by using (+) sign appears in the pop-up. Click the find button and refined list based on your search criteria will appear in grid

4. If you want to remove a search criteri then use (-) sign appearing next to criteria

5. Use reset to remove all search criteria

+ Refresh

A refresh or reload will cause the browser to re-check with the web server to make sure it has the latest copy of the web page you are viewing.to refresh the grid

1. Click the Refresh button in your page. Grid data will be reloaded from the server to ensure that you are viewing the latest copy.

+ Export

Export allow you to To export the data in excel format on user system. If filter is applied then only filtered record from the screen will be exported to excel, to export

1. Click the export button in your page. A window will appear to save or open the excel.

2. based on your selection file will be saved on system.
