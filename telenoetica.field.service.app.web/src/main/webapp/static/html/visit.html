This Section contains 4 pages, one for each visit from where user can submit visit information. This web page can be used by field service engineer only when he is not able to submit the information from his android application. There are 4 kinds of visits supported by field application system and 4 pages for the same:

+ Routine Visit

Routine Visit page contains following fields


Site ID : Site where visit is happening.Autofill text with pattern "XX/XX/XX/NNN".Click on the input field and start typing, application will suggest you the possible values

Access Code : Num- N digits

Diesel Level in T1: Num- 0 TO 6000

Diesel Level in T2: Num- 0 TO 6000

Run-Hour of Gen1 : Num- 0 TO 30000

Run-Hour of Gen2 : Num- 0 TO 30000

Voltage: N-R volts : Num- 0 TO 400

Voltage: N-Y volts : Num- 0 TO 400

Voltage: N-B volts : Num- 0 TO 400

Load: R phase Amps : Num- 0 TO 999

Load: Y phase Amps : Num- 0 TO 999

Load: B phase Amps : Num- 0 TO 999

Rectifier O/P voltage : Num- 0 TO 999

Rectifier O/P current Amp : Num- 0 TO 999

Battery capacity V/AH : Num- 0 TO 50

Battery capacity AH : Num- 0 TO 2000

ATS Functional : Yes or No

ATS Synchronising : Yes or No

DRN Book at site : Yes or No

Diesel log-book maintained : Yes or No

Aircon of shelter-1 cooling : Yes or No or Not applicable

Aircon of shelter-2 cooling : Yes or No or Not applicable

Aircon of shelter-3 cooling : Yes or No or Not applicable

Aircon of shelter-4 cooling : Yes or No or Not applicable

+ Diesel Visit

Diesel Visit page contains following fields


Site ID : Site where visit is happening.Autofill text with pattern "XX/XX/XX/NNN".Click on the input field and start typing, application will suggest you the possible values

Access Code : Num- N digits

DRN Number : Num- Length 7 digits

Diesel as transfer or bulk supply : Transfer / Bulk

If Transfer, then from which site? : Site from where transfer is happening.Autofill text with pattern "XX/XX/XX/NNN".Click on the input field and start typing, application will suggest you the possible values


If Bulk, then name of vendor? : Browse & select

Diesel Level in T1 before receipt : Num- 0 TO 6000

Diesel Level in T2 before receipt : Num- 0 TO 6000

Diesel received (Ltrs) : Num- 0 TO 6000

Run-Hour of Gen1 : Num- 0 TO 30000

Run-Hour of Gen2 : Num- 0 TO 30000

DRN Book at site : Yes or No

Diesel log-book maintained : Yes or No

PHCN Installed : Yes or No

PHCN Hrs per day : Num- 0 TO 24

Hybrid/PIU installed : Yes or No

Hybrid/PIU hrs per Day : Num- 0 TO 24

+ Callout Visit

Diesel Visit page contains following fields


Site ID : Site where visit is happening.Autofill text with pattern "XX/XX/XX/NNN".Click on the input field and start typing, application will suggest you the possible values

Access Code : Num- N digits

Call-out CSR or TT number: Number

Time when complain received : Date and Time

Time when reached to site :Date and Time,should be after above incident.

Time when fault resolved :Date and Time, should be after above 2 incidents.

Customer1 Impacted :Browse & select

Customer2 Impacted :Browse & select

Customer3 Impacted :Browse & select

Customer4 Impacted :Browse & select

Main Category of fault :Browse & select

Equipment/ Component that caused fault :Browse & select

Equipment/ Componenet repaired :Browse & select

Equipment/ Componenet replaced :Browse & select

Is the fix/ resolution temporary or permanent? :Temporary or Permanent

Actions required for permanent resolution :Alphanumeric - 0 TO 50
